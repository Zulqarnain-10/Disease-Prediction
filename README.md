# Disease Prediction

*A machine-learning model that predicts disease likelihood from patient health data.*

## Problem

Early risk signals help prioritize care. Predicting disease from structured health indicators supports faster, data-driven screening and decision support.

## Approach

Cleaned and preprocessed the health dataset, engineered features, trained a classification model, and evaluated it with standard performance metrics.

## Tech Stack

`Python` · `NumPy` · `pandas` · `Jupyter`

## How to Run

Open `Disease_Prediction.ipynb` in Jupyter and run all cells.
